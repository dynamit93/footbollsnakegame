# Football Snake Game

Soccer-themed grid game with Snake-style movement and online 1v1 multiplayer (React + Socket.IO).

## Quick start

Requires Node.js 20+.

```bash
npm install
npm run dev
```

Open the Vite URL (usually [http://localhost:5173](http://localhost:5173)). Run `npm run build` for production builds.
